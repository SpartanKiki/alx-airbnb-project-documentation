# User Stories for Airbnb Clone Backend

## 1. User Management

### User Registration
- As a **guest**, I want to create an account so that I can book properties.
- As a **host**, I want to create an account so that I can list my properties.

### Login & Authentication
- As a **user**, I want to log in securely so that I can access my account.
- As a **user**, I want to log in with Google or Facebook so that I can sign in faster.

### Profile Management
- As a **user**, I want to update my profile so that my information is accurate.
- As a **user**, I want to upload a profile picture so that my identity feels personal.

---

## 2. Property Listings Management

### Add Listings
- As a **host**, I want to add a property listing so that I can rent it out.
- As a **host**, I want to upload images for a listing so that my property is appealing.

### Edit/Delete Listings
- As a **host**, I want to edit a property listing so that I can update information.
- As a **host**, I want to delete a listing so that I can remove properties I no longer rent.

---

## 3. Search and Filtering

- As a **guest**, I want to search for properties by location so that I can find stays where I need them.
- As a **guest**, I want to filter properties by price so that I stay within my budget.
- As a **guest**, I want to filter by number of guests so that the property fits my needs.
- As a **guest**, I want to filter by amenities so that I can find properties with required features.

---

## 4. Booking Management

### Create Booking
- As a **guest**, I want to book a property for selected dates so that I can reserve my stay.
- As a **guest**, I want to get a warning if the dates are unavailable so that I avoid double bookings.

### Cancel Booking
- As a **guest**, I want to cancel a booking so that I can change my plans.
- As a **host**, I want to cancel a booking in emergencies so that I manage availability.

### Booking Status
- As a **user**, I want to view booking status so that I know if my stay is confirmed.

---

## 5. Payments

- As a **guest**, I want to pay for a booking so that my reservation is confirmed.
- As a **host**, I want to receive payouts automatically after a guest completes their stay.
- As a **guest**, I want to see total costs before paying so that I avoid surprises.

---

## 6. Reviews and Ratings

- As a **guest**, I want to leave a review so that I can share my experience.
- As a **host**, I want to respond to reviews so that I can address guest feedback.
- As a **guest**, I want to review only properties I stayed in so that reviews are fair.

---

## 7. Notifications

- As a **user**, I want to receive a confirmation email after booking so that I know it's saved.
- As a **user**, I want notifications for cancellations so that I stay informed.
- As a **host**, I want payment notifications so that I know payouts are processed.

---

## 8. Admin Dashboard

- As an **admin**, I want to manage users so that I keep the platform safe.
- As an **admin**, I want to manage listings so that inappropriate content is removed.
- As an **admin**, I want to view bookings so that I can resolve disputes.
- As an **admin**, I want to monitor payments so that the system remains consistent.
